# Contributing Guidelines

Thank you for your interest in contributing to this project!

## How to Contribute

1. **Reporting Bugs**: Feel free to open an issue describing the bug, including steps to reproduce it, expected behavior, and screenshots if applicable.
2. **Suggesting Enhancements**: Open an issue detailing the enhancement you'd like to see, along with the reasoning behind it.
3. **Pull Requests**:
   - Fork the repository.
   - Create a feature branch.
   - Make your changes.
   - Ensure the code follows consistent styling and contains no errors.
   - Submit a pull request detailing your changes.

## Code of Conduct

Please note that this project is released with a Contributor Covenant Code of Conduct. By participating in this project, you agree to abide by its terms.
