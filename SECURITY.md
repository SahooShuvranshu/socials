# Security Policy

## Supported Versions

Only the latest version of the repository is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

Please do not open a public issue on GitHub for security vulnerabilities. Instead, report security bugs or concerns directly via email to:

**sahooshuvranshusekhar@gmail.com**

We will acknowledge receipt of your vulnerability report within 48 hours and work with you to resolve the concern.
